Quick Start

To start using one of Software Collections (SCLs), all you need to do is follow these steps:

  1. Find a collection you are interested in. You can browse through available collections in our catalogue. We will use perl516 collection as an example
  2. Download and install the collection repository RPM file for your system (EPEL 6 on x86_64 in our case): yum install perl516-epel-6-x86_64-1-1.noarch.rpm
  3. Install main SCL package with name identical to the name of the collection: yum install perl516
  4. Start using newly installed collection: scl enable perl516 perl

For more usage examples and documentation of more complex scenarios see full Software Collections Guide
