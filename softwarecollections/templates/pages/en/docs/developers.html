Software Collections for Developers

Collection prefix

The collection prefix is defined by the macro _scl_prefix. In copr must be the prefix unique, therefore it's based on name of a project and a username e.g. ruby193-username. Copr has preset of the installation path into /opt/sclo. The installation path and the collection prefix can be overriden. In case of local builds can be macros redefined in /etc/rpm/macros.scl, namely scl_basedir and scl_vendor, or define in every specfile. The new definition of intallation path is not recommended in scl.org repositories.
