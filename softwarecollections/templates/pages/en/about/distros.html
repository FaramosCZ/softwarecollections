Software Collections for Distros

All collection on SCL.org are using the prefix (not distribution specific): /opt/sclo

Each distribution should register their own directory in the /opt directory and use it for their specific collections. The registration can be done by bug request via Linux Foundation bugzilla, under product LANANA.

The prefix is defined in the scl-utils package. You can find more about scl-utils macros here.

When building in Copr, it is easier to use a package with redefiniton of the prefix. For more information about building in Copr see: https://fedorahosted.org/copr/wiki/ScreenshotsTutorial
